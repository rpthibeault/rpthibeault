### Projects

I am currently building [umbra](https://github.com/rpthibeault/umbra), a hobby operating system written completely from scratch with C and Assembly. The current features are a multiboot2-compliant bootloader, and a simple shell. I am currently focusing development on kernel subsystem implementation e.g. memory management, task scheduling, and so on.

For my Honours Project, I developed a Parallelized Code Generator [found here](https://github.com/rpthibeault/autopar) using C++, LLVM, and OpenMP. I used task-based parallelism techniques to achieve 2.5x+ faster runtimes on sequential programs.
